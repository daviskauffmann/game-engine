# Project Kilonova

## Getting Started

### Dependencies

- SDL2
- SDL2_image
- SDL2_mixer
- GLEW
- GLM
- Assimp
- OpenAL

### Download

```sh
git clone https://github.com/daviskauffmann/project-kilonova.git
cd project-kilonova
git submodule update --init
```

### Build

```sh
make
```

### Build & Run

```sh
make run
```

### Cleanup

```sh
make clean
```
